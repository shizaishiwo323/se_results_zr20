可以，这个任务我帮你整理成一套**可以直接写进方法章节 + 后续转代码的震电模拟方案**。先说一个关键判断：

**你的反应输运数据不能直接“画震电波图”，中间必须经过一层 Schakel & Smeulders 2010 的震电转换系数计算。**

也就是：

[
\text{反应输运参数}
\rightarrow
\phi(t_d), k_0(t_d), \alpha_\infty(t_d), C(t_d), pH(t_d)
\rightarrow
k(\omega),L(\omega),\sigma(\omega)
\rightarrow
R_E(\omega,\theta;t_d), T_{TM}(\omega,\theta;t_d)
\rightarrow
u_r(z,t;t_d),u_t(z,t;t_d)
]

其中 (t_d) 表示**溶解/反应输运时间**，不是声波传播时间；而震电波图横轴里的 (t) 是**声波激发后的微秒级波形时间**。这两个时间尺度一定要分开。

---

## 1. 先定义你的输入参数

对于每一个溶解时刻 (t_d)，你已经有：

[
\phi(t_d),\quad k_0(t_d),\quad \alpha_\infty(t_d),\quad c_H(t_d)
]

其中：

| 符号                   | 含义                          | 来自你的数据          |
| -------------------- | --------------------------- | --------------- |
| (\phi(t_d))          | 孔隙率                         | Porosity        |
| (k_0(t_d))           | 静态渗透率，单位必须转成 (\mathrm{m^2}) | Permeability_mD |
| (\alpha_\infty(t_d)) | 高频曲折度 / 迂曲度                 | Tortuosity      |
| (c_H(t_d))           | 出水氢离子浓度                     | OutletHConc     |
| (C(t_d))             | 电解质浓度                       | 需要由离子浓度估计       |
| (pH(t_d))            | 酸碱度                         | 由 (c_H) 计算      |

渗透率单位换算：

[
k_0[\mathrm{m^2}]
=================

k_0[\mathrm{mD}]\times 9.869233\times 10^{-16}
]

如果 (c_H) 的单位是 (\mathrm{mol/L})，则：

[
pH(t_d)=-\log_{10} c_H(t_d)
]

如果 (c_H) 的单位是 (\mathrm{mol/cm^3})，则先换算：

[
c_H[\mathrm{mol/L}]
===================

1000,c_H[\mathrm{mol/cm^3}]
]

再算：

[
pH(t_d)=-\log_{10}\left(1000c_H(t_d)\right)
]

这里要特别注意：**Schakel & Smeulders 2010 的 zeta 电位公式用的是电解质浓度 (C) 和 pH，不是只用 (H^+) 浓度。**如果你现在只有 (H^+)，第一版可以设：

[
C(t_d)=C_{\mathrm{bg}}+c_H(t_d)
]

其中 (C_{\mathrm{bg}}) 是背景电解质浓度，比如论文表 1 里用的 (10^{-3}\ \mathrm{mol/L})。但这一步属于模型假设，论文里要写清楚。

---

## 2. Schakel & Smeulders 2010 的严格计算主线

Schakel & Smeulders 做的是：一个流体中的入射 P 波打到流体/多孔介质界面，然后转换出流体侧反射 EM 波和多孔介质侧透射 TM-EM 波。作者明确给出两个转换系数：(R_E) 表示震电反射系数，(T_{TM}) 表示 TM 型震电透射系数；它们通过完整边界值问题求解，而不是经验拟合。

### 2.1 动态渗透率

对每个溶解时刻 (t_d)，先计算特征过渡角频率：

[
\omega_t(t_d)=
\frac{\eta}{\alpha_\infty(t_d),k_0(t_d),\rho_f}
]

其中：

| 符号              | 含义                  |
| --------------- | ------------------- |
| (\omega_t)      | 黏性主导流动向惯性主导流动过渡的角频率 |
| (\eta)          | 孔隙流体黏度              |
| (\rho_f)        | 孔隙流体密度              |
| (k_0)           | 静态渗透率               |
| (\alpha_\infty) | 曲折度                 |

然后计算动态渗透率：

[
k(\omega;t_d)
=============

k_0(t_d)
\left[
\sqrt{1+i\frac{\omega}{\omega_t(t_d)}\frac{M(t_d)}{2}}
+i\frac{\omega}{\omega_t(t_d)}
\right]^{-1}
]

这里：

| 符号          | 含义                        |
| ----------- | ------------------------- |
| (k(\omega)) | 动态渗透率                     |
| (\omega)    | 入射声波角频率，(\omega=2\pi f)   |
| (M)         | similarity parameter，相似参数 |
| (f)         | 入射声波频率                    |

Schakel 论文附录 A 给出动态渗透率、(\omega_t) 和 (M) 的关系。

---

### 2.2 孔隙几何尺度 (\Lambda)

论文里 (M)、(k_0)、(\alpha_\infty)、(\Lambda) 满足：

[
M(t_d)=
\frac{8\alpha_\infty(t_d)k_0(t_d)}
{\phi(t_d)\Lambda^2(t_d)}
]

其中：

| 符号        | 含义                                                |
| --------- | ------------------------------------------------- |
| (\Lambda) | weighted pore volume-to-surface ratio，加权孔隙体积—表面积比 |
| (M)       | 相似参数                                              |
| (\phi)    | 孔隙率                                               |

如果你暂时没有可靠的 (\Lambda)，可以先按 Schakel 表 1 那样取：

[
M=1
]

于是：

[
\Lambda(t_d)
============

\sqrt{
\frac{8\alpha_\infty(t_d)k_0(t_d)}
{\phi(t_d)}
}
]

但你这里有几何切片和 SurfaceArea，后面更高级的做法是用几何直接估计：

[
\Lambda(t_d)
\approx
\frac{V_{\mathrm{pore}}(t_d)}
{S_{\mathrm{solid-fluid}}(t_d)}
]

或者用一个比例因子修正为：

[
\Lambda(t_d)=\chi
\frac{V_{\mathrm{pore}}(t_d)}
{S_{\mathrm{solid-fluid}}(t_d)}
]

其中 (\chi) 是校正系数。这个很适合做你自己的贡献，因为它把反应输运几何演化真正接进震电模型了。

---

### 2.3 zeta 电位

Schakel & Smeulders 使用 Pride & Garambois 2005 的半经验公式：

[
\zeta(t_d)
==========

\left[
0.010+0.025\log_{10}C(t_d)
\right]
\frac{pH(t_d)-2}{5}
]

其中：

| 符号      | 含义                        |
| ------- | ------------------------- |
| (\zeta) | zeta 电位，单位 V              |
| (C)     | 电解质浓度，单位 (\mathrm{mol/L}) |
| (pH)    | 酸碱度                       |

这一步非常重要，因为 (L(\omega)) 正比于 (\zeta)，所以你的氢离子浓度变化会通过 pH 影响震电耦合强度。Schakel 论文的敏感性分析也指出 pH 对震电能量系数近似线性增强。

---

### 2.4 Debye 长度

[
d(t_d)
======

\left[
\sum_{l=1}^{N_i}
\frac{(e z_l)^2N_l(t_d)}
{\varepsilon_0\varepsilon_f k_BT}
\right]^{-1/2}
]

其中：

| 符号              | 含义                                 |
| --------------- | ---------------------------------- |
| (d)             | Debye 长度                           |
| (e)             | 元电荷                                |
| (z_l)           | 第 (l) 种离子的价态                       |
| (N_l)           | 第 (l) 种离子的数密度，单位 (\mathrm{m^{-3}}) |
| (\varepsilon_0) | 真空介电常数                             |
| (\varepsilon_f) | 孔隙流体相对介电常数                         |
| (k_B)           | Boltzmann 常数                       |
| (T)             | 温度                                 |

如果用二元对称电解质：

[
z_1=1,\quad z_2=-1,\quad N_1=N_2=N_A\cdot 1000C
]

其中 (N_A) 是 Avogadro 常数，(C) 单位是 (\mathrm{mol/L})。

---

### 2.5 电动耦合系数 (L(\omega))

Schakel & Smeulders 附录 A 给出：

[
L(\omega;t_d)
=============

*

\frac{\phi(t_d)}{\alpha_\infty(t_d)}
\frac{\varepsilon_0\varepsilon_f\zeta(t_d)}{\eta}
\left(
1-2\frac{d(t_d)}{\Lambda(t_d)}
\right)
\left[
1+
2i\frac{\omega}{M(t_d)\omega_t(t_d)}
\left(
1-2\frac{d(t_d)}{\Lambda(t_d)}
\right)^2
\left(
1+d(t_d)\sqrt{\frac{i\omega\rho_f}{\eta}}
\right)^2
\right]^{-1/2}
]

其中：

| 符号          | 含义     |
| ----------- | ------ |
| (L(\omega)) | 电动耦合系数 |
| (\eta)      | 孔隙流体黏度 |
| (\rho_f)    | 孔隙流体密度 |
| (i)         | 虚数单位   |

这个公式是你整套工作的核心，因为你的 (\phi,k_0,\alpha_\infty,pH,C) 都会汇入 (L(\omega))。Schakel 的敏感性讨论明确说，渗透率和曲折度主要通过 (\Lambda) 和 (L(\omega)) 影响震电转换；孔隙率也通过 (L(\omega)) 近似线性影响震电转换。

---

### 2.6 动态电导率 (\sigma(\omega))

[
\sigma(\omega;t_d)
==================

\frac{\phi(t_d)\sigma_f(t_d)}{\alpha_\infty(t_d)}
\left[
1+
\frac{2\left(C_{\mathrm{em}}(t_d)+C_{\mathrm{os}}(\omega;t_d)\right)}
{\sigma_f(t_d)\Lambda(t_d)}
\right]
]

其中：

[
\sigma_f(t_d)
=============

\sum_{l=1}^{N_i}
(ez_l)^2 b_l N_l(t_d)
]

[
C_{\mathrm{em}}(t_d)
====================

2d(t_d)
\sum_{l=1}^{N_i}
(ez_l)^2 b_l N_l(t_d)
\left[
\exp
\left(
-\frac{ez_l\zeta(t_d)}{2k_BT}
\right)-1
\right]
]

[
C_{\mathrm{os}}(\omega;t_d)
===========================

\frac{(\varepsilon_0\varepsilon_f)^2\zeta^2(t_d)}
{2d(t_d)\eta}
P(t_d)
\left[
1+
\frac{2}{P(t_d)}
d(t_d)\sqrt{\frac{i\omega\rho_f}{\eta}}
\right]^{-1}
]

[
P(t_d)
======

\frac{8k_BT d^2(t_d)}
{\varepsilon_0\varepsilon_f\zeta^2(t_d)}
\sum_{l=1}^{N_i}
N_l(t_d)
\left[
\exp
\left(
-\frac{ez_l\zeta(t_d)}{2k_BT}
\right)-1
\right]
]

这些是 Schakel 附录 A 里动态电导率、流体电导率、表面过量电导和电渗过量电导的表达式。

## 3. 计算 (R_E) 和 (T_{TM})

### 3.1 先计算 Biot 弹性参数

对每个 (t_d)，用 (\phi(t_d)) 更新：

[
A(t_d),\quad Q(t_d),\quad R(t_d)
]

它们由固体颗粒体积模量 (K_s)、孔隙流体体积模量 (K_f)、骨架体积模量 (K_b)、剪切模量 (G) 和孔隙率 (\phi) 决定。Schakel 正文的式 8–10 给出 (A,Q,R)，并说明它们是 Biot-Willis 广义弹性系数。

---

### 3.2 计算有效介电常数

[
\bar{\varepsilon}(\omega;t_d)
=============================

## \varepsilon(t_d)

i\frac{\sigma(\omega;t_d)}{\omega}
+
i
\frac{\eta L^2(\omega;t_d)}
{\omega k(\omega;t_d)}
]

其中：

| 符号                  | 含义             |
| ------------------- | -------------- |
| (\bar{\varepsilon}) | 有效电容率 / 有效介电参数 |
| (\varepsilon)       | 多孔介质体电容率       |
| (\sigma(\omega))    | 动态电导率          |

Schakel 式 20 明确把 (\bar{\varepsilon}) 定义成这个形式，用来把导电、电动耦合和介电效应统一到波速计算里。

---

### 3.3 计算有效密度

[
\rho_{12}(\omega;t_d)
=====================

\rho_f
\left[
1+i\frac{\eta}
{\omega\rho_f k(\omega;t_d)}
\right]
]

[
\rho_{11}(\omega;t_d)
=====================

(1-\phi(t_d))\rho_s-\rho_{12}(\omega;t_d)
]

[
\rho_{22}(\omega;t_d)
=====================

\phi(t_d)\rho_f-\rho_{12}(\omega;t_d)
]

然后引入电动耦合修正项：

[
E_K(\omega;t_d)
===============

\frac{\eta^2L^2(\omega;t_d)}
{k^2(\omega;t_d)\bar{\varepsilon}(\omega;t_d)\omega^2}
]

[
\bar{\rho}*{11}=\rho*{11}-E_K
]

[
\bar{\rho}*{12}=\rho*{12}+E_K
]

[
\bar{\rho}*{22}=\rho*{22}-E_K
]

Schakel 的一个重要处理就是把黏性耦合和电动耦合都并入频率相关的有效密度里。

---

### 3.4 计算四类波的慢度

慢度 (s) 是波速的倒数：

[
s=\frac{1}{v}
]

#### 快 P 波和慢 P 波

令：

[
P=A+2G
]

[
d_0=
\bar{\rho}*{11}\bar{\rho}*{22}
------------------------------

\bar{\rho}_{12}^2
]

[
d_1=
----

\left(
P\bar{\rho}*{22}
+
R\bar{\rho}*{11}
----------------

2Q\bar{\rho}_{12}
\right)
]

[
d_2=PR-Q^2
]

则快 P 波 (P_f) 和慢 P 波 (P_s) 的平方慢度为：

[
s_l^2(\omega;t_d)
=================

-\frac{d_1}{2d_2}
\pm
\frac{1}{2}
\sqrt{
\left(\frac{d_1}{d_2}\right)^2
------------------------------

4\frac{d_0}{d_2}
}
]

其中：

[
l=P_f,P_s
]

Schakel 式 24–25 给出了这个色散关系。

#### TM 电磁波和 S 波

横波也使用同样形式：

[
s_l^2(\omega;t_d)
=================

-\frac{d_1}{2d_2}
\pm
\frac{1}{2}
\sqrt{
\left(\frac{d_1}{d_2}\right)^2
------------------------------

4\frac{d_0}{d_2}
}
]

但这里：

[
l=TM,S
]

并使用 Schakel 式 30 中的 (d_0,d_1,d_2)。

---

### 3.5 计算幅值比

对纵波：

[
\beta_m(\omega;t_d)
===================

\frac{\hat{\phi}^{m}*{f}}
{\hat{\phi}^{m}*{s}}
====================

\frac{
\bar{\rho}_{11}(\omega;t_d)
---------------------------

P s_m^2(\omega;t_d)
}
{
Q s_m^2(\omega;t_d)
-------------------

\bar{\rho}_{12}(\omega;t_d)
}
]

[
\alpha_m(\omega;t_d)
====================

\frac{\hat{\phi}^{m}*{E}}
{\hat{\phi}^{m}*{s}}
====================

\frac{\eta L(\omega;t_d)}
{k(\omega;t_d)\bar{\varepsilon}(\omega;t_d)}
\left[
1-\beta_m(\omega;t_d)
\right]
]

其中：

[
m=P_f,P_s
]

对横波：

[
\beta_n(\omega;t_d)
===================

\frac{\hat{\psi}^{n}*{f}}
{\hat{\psi}^{n}*{s}}
====================

\frac{
G s_n^2(\omega;t_d)
-------------------

(1-\phi(t_d))\rho_s
}
{\phi(t_d)\rho_f}
]

[
\alpha_n(\omega;t_d)
====================

\frac{\hat{\psi}^{n}*{E}}
{\hat{\psi}^{n}*{s}}
====================

\frac{
\eta L(\omega;t_d)
}
{
k(\omega;t_d)
\left[
\bar{\varepsilon}(\omega;t_d)
-----------------------------

s_n^2(\omega;t_d)
\right]
}
\left[
1-\beta_n(\omega;t_d)
\right]
]

其中：

[
n=TM,SV
]

这些幅值比对应 Schakel 式 36–39，用来把机械位移势和电势联系起来。

### 3.6 建立边界矩阵并求系数

未知量向量为：

[
\mathbf{x}
==========

\left[
R_E,\ R_M,\ T_{Pf},\ T_{Ps},\ T_{TM},\ T_{SV}
\right]^T
]

其中：

| 符号       | 含义                          |
| -------- | --------------------------- |
| (R_E)    | 震电反射系数，入射声波转换成流体侧反射 EM 波的系数 |
| (R_M)    | 机械反射系数，普通声波反射系数             |
| (T_{Pf}) | 快 P 波透射系数                   |
| (T_{Ps}) | 慢 P 波透射系数                   |
| (T_{TM}) | TM 型 EM 波透射系数               |
| (T_{SV}) | SV 波透射系数                    |

Schakel 的系数定义是：

[
R_E=
\frac{\tilde{\psi}^{R}*{fl}}
{\tilde{\phi}^{I}*{fl}}
]

[
T_{TM}
======

\frac{\tilde{\psi}^{TM}*{s}}
{\tilde{\phi}^{I}*{fl}}
]

论文式 45 给出了这些反射/透射系数的定义，式 46 说明它们由一个 (6\times 6) 线性系统求解。

线性系统为：

[
\mathbf{A}(\omega,\theta;t_d)\mathbf{x}
=======================================

\mathbf{b}
]

[
\mathbf{b}
==========

\left[
k_3^{fl},
\phi\rho_{fl},
0,
0,
0,
0
\right]^T
]

最终：

[
\mathbf{x}
==========

\mathbf{A}^{-1}\mathbf{b}
]

于是：

[
R_E(\omega,\theta;t_d)=x_1
]

[
T_{TM}(\omega,\theta;t_d)=x_5
]

## 4. 你要画的第一类图：某一溶解时刻的震电波图

刘老师论文的波形图本质上是：固定一个溶解时刻 (t_d=t_d^\ast)，沿电极接收线取很多位置 (z_r)，然后画：

[
u(z_r,t;t_d^\ast)
]

这里：

| 符号         | 含义                       |
| ---------- | ------------------------ |
| (z_r)      | 电极相对界面的距离                |
| (t)        | 声波激发后的传播时间，单位通常是 (\mu s) |
| (t_d^\ast) | 选定的溶解过程时刻                |
| (u)        | 电势或电场响应                  |

刘老师论文使用的是频率—波数积分形式：

[
u_r(x,z,t;t_d)
==============

\iint
A(k,\omega)
R_E(k,\omega;t_d)
\exp
\left[
i
\left(
k_z^i z_s
---------

k_z^{Er}z
+
kx
--

\omega t
\right)
\right]
dk,d\omega
]

[
u_t(x,z,t;t_d)
==============

\iint
A(k,\omega)
T_E(k,\omega;t_d)
\exp
\left[
i
\left(
k_z^i z_s
+
k_z^{Et}z
+
kx
--

\omega t
\right)
\right]
dk,d\omega
]

其中：

| 符号            | 含义                               |
| ------------- | -------------------------------- |
| (u_r)         | 流体侧接收的反射界面 EM 电势                 |
| (u_t)         | 多孔介质侧接收的透射界面 EM 电势               |
| (x,z)         | 接收点坐标                            |
| (z_s)         | 声源到界面的距离                         |
| (k)           | 水平波数                             |
| (\omega)      | 角频率                              |
| (k_z^i)       | 入射声波垂向波数                         |
| (k_z^{Er})    | 流体中 EM 波垂向波数                     |
| (k_z^{Et})    | 多孔介质中 EM 波垂向波数                   |
| (A(k,\omega)) | 声源频率—波数谱                         |
| (R_E)         | Schakel 计算的震电反射系数                |
| (T_E)         | Schakel 计算的震电透射系数，第一版可取 (T_{TM}) |

刘老师论文就是用这个理论模型生成有限偏移距 VSEP 波形，并指出接收电极沿测线穿过界面时，所有信号在同一个 (T_0) 到达；这个 (T_0) 是声波从声源传播到界面的时间，而不是 EM 波传播时间。

---

### 4.1 第一版零偏移距设置

你说第一版先做无偏移距，所以：

[
D=0
]

[
x=0
]

声源在：

[
(0,z_s)
]

接收电极在：

[
(0,z_r)
]

声波垂直入射界面：

[
\theta=0
]

到时：

[
T_0=
\frac{z_s}{V_f}
]

其中 (V_f) 是上覆流体中的声速。

但这里有一个很容易踩坑的地方：**Schakel 的单个平面波模型在 (\sin\theta=0) 时 (R_E) 和 (T_{TM}) 为 0**，这是由它的边界条件和势函数定义导致的。论文也明确说 (\sin\theta=0) 时没有反射 EM 波生成，而 (\sin\theta=1) 是掠入射且震电转换也为 0。

所以你不能直接用：

[
R_E(\theta=0)
]

去代表刘老师图里的零偏移距震电波形。正确做法是：

[
\text{用刘老师的有限宽度声源频率—波数积分}
]

也就是虽然主入射方向是垂直的，但有限宽度声源包含一组非零水平波数 (k)，这些 (k) 分量会贡献界面 EM 响应。

## 8. 你的论文方法部分可以这样写

[

\rightarrow
\left[
\phi,k_0,\alpha_\infty,C,pH
\right](t_d)
\rightarrow
\left[
k(\omega),L(\omega),\sigma(\omega)
\right](t_d)

\right](\omega,\theta,t_d)

}
]

文字版可以写成：

> At each dissolution time, the porosity, permeability, tortuosity and outlet hydrogen concentration obtained from the reactive transport simulation were mapped into the electrokinetic coefficients of Pride’s seismoelectric theory. The dynamic permeability, electrokinetic coupling coefficient and dynamic conductivity were computed following Schakel and Smeulders. The seismoelectric reflection coefficient (R_E) and TM-mode transmission coefficient (T_{TM}) were then obtained by solving the coupled boundary-value problem at a fluid–porous medium interface. Finally, the time-domain interface EM waveforms were synthesized using the finite-width acoustic source formulation of Liu et al.

## 9. 最关键的实现提醒

你现在 Excel 里的孔隙率后期接近甚至达到：

[
\phi\approx 1
]

这在 Biot 多孔介质理论里会让骨架接近消失，很多参数会变得病态。所以数值计算时需要设置有效范围，比如：

[
0.05 < \phi < 0.95
]

超过这个范围的点可以标注为：

```text
invalid for poroelastic seismoelectric model
```

或者只画到：

[
\phi \le 0.95
]

这个很重要，不然到后期你的 (R_E)、(T_{TM}) 可能会数值爆炸，不是物理现象，而是模型适用性失效。
